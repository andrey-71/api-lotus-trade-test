# Lotus Trade API
___
API для тестового задания


## Ссылки:
* URL: https://api.lotus.test-online.online
___


## Технологии:
* REST API;
* Node.js;
* Express.js;
* MongoDB.
___

## Инструкция по локальному развертыванию:
* Клонировать репозиторий:
  ```
    git@github.com:andrey-71/api-lotus-trade-test.git
  ```
* Установить зависимости
  ```
  npm i
  ```
* Запуск проекта
  ```
  npm run start
  ```
